# История создания  

## Этап 1. Создание проекта  

1. Создание структуры:  

```
slay/
├── backend/                <- я внутри нахожусь
├── docs/
│   ├── plantuml_diagrams/
│   └── other_docs/
└── frontend/
```  

2. Создание виртуального окружения  

```bash
python -m venv venv
source venv/Scripts/activate
```  

3. Установка Django и DRF  

```bash
pip install django djangorestframework
```  

_(`requirements.txt` — `pip freeze > requirements.txt`)_  

4. Создание проекта  

```bash
django-admin startproject slay_api .
```  

5. Создание 4 приложений  

```bash
python manage.py startapp users
python manage.py startapp catalog
python manage.py startapp cart
python manage.py startapp orders
```  

6. Регистрация приложений в `settings.py`  
Открываю `backend/slay_api/settings.py` и добавляю:  

```python
INSTALLED_APPS = [
    ...
    'rest_framework',
    'users.apps.UsersConfig',
    'catalog.apps.CatalogConfig',
    'cart.apps.CartConfig',
    'orders.apps.OrdersConfig',
]
```  

7. Применение миграций и первый запуск  

```bash
# Создать таблицы в БД (для пользователей, сессий и т.д.)
python manage.py migrate

# Запустить сервер
python manage.py runserver
```

## Этап 6. Приложение `cart`  

- Реализованы endpoints корзины в `views.py`:  
  - `GET /api/cart/`  
  - `POST /api/cart/items/`  
  - `PATCH /api/cart/items/{id}/`  
  - `DELETE /api/cart/items/{id}/`  
  - `DELETE /api/cart/clear/`  

- Гостевая корзина хранится в Django session.  
- Авторизованная корзина хранится в БД.  
- При логине гостевая корзина сливается с корзиной пользователя.  
- В `users/views.py` изменен только `LoginView`, чтобы после успешного логина вызвать `merge` корзины. Формат ответа логина остается прежним.

## Этап 9. Фронтенд  

- Реализовала CORS:  
`pip install django-cors-headers`  

```python
# settings.py
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'rest_framework',
    'corsheaders',  # для CORS
    'rest_framework_simplejwt',
    'users.apps.UsersConfig',
    'catalog.apps.CatalogConfig',
    'cart.apps.CartConfig',
    'orders.apps.OrdersConfig',
    'drf_spectacular',  # для генерации документации
]

MIDDLEWARE = [
    'django.middleware.security.SecurityMiddleware',
    'django.contrib.sessions.middleware.SessionMiddleware',
    'corsheaders.middleware.CorsMiddleware',  # для CORS. обязательно выше CommonMiddleware
    'django.middleware.common.CommonMiddleware',
    'django.middleware.csrf.CsrfViewMiddleware',
    'django.contrib.auth.middleware.AuthenticationMiddleware',
    'django.contrib.messages.middleware.MessageMiddleware',
    'django.middleware.clickjacking.XFrameOptionsMiddleware',
]

CORS_ALLOWED_ORIGINS = [  # адрес, с которого разрешены запросы
    'http://localhost:5173',
]
```
