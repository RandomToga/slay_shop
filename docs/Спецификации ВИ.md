# Спецификация вариантов использования системы
### 1. Зарегистрироваться (Пользователь)

| Поле | Описание |
| :--- | :--- |
| **Название** | **Зарегистрироваться** |
| **Приоритет** | Высокий |
| **Основной актор** | Пользователь |
| **Заинтересованные лица** | Пользователь, система интернет-магазина |
| **Предусловие** | Пользователь находится на странице регистрации. |
| **Основной успешный поток событий** | 1. Пользователь вводит email и пароль.<br>2. Система проверяет корректность данных и уникальность email.<br>3. Система создает новую учетную запись.<br>4. Система уведомляет об успешной регистрации. |
| **Постусловие** | Учетная запись создана. Пользователь может авторизоваться. |
| **Альтернативный поток №1** | Введенный email уже занят или данные не прошли проверку. Система выводит сообщение об ошибке. |
| **Постусловие №1** | Учетная запись не создана. Пользователь остается на форме регистрации. |
